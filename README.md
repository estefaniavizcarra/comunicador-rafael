# Comunicador de Rafael 🗣️

Tablero de comunicación (CAA) hecho a la medida de Rafael. Funciona en el iPad como una app,
con pictogramas ARASAAC, voz en español, y editor para agregar fotos y palabras.

Este archivo (`index.html`) es **todo lo que se necesita** — ya trae los pictogramas
incrustados y funciona sin internet una vez cargado.

---

## 📲 Cómo publicarlo en GitHub Pages (para abrirlo en el iPad)

La idea: subir esta carpeta a GitHub, activar "Pages", y GitHub te da un **enlace**.
Ese enlace lo abres en **Safari** del iPad → funciona → lo agregas al inicio como app.

### 1. Crear el repositorio
1. Entra a **https://github.com** (crea una cuenta gratis si no tienes).
2. Arriba a la derecha: **+ → New repository**.
3. Nombre: `comunicador-rafael`
4. Déjalo **Public** (necesario para Pages gratis) → **Create repository**.

### 2. Subir el archivo
1. En el repositorio nuevo: **Add file → Upload files**.
2. Arrastra el archivo **`index.html`** de esta carpeta.
3. Abajo: **Commit changes**.

### 3. Activar GitHub Pages
1. En el repositorio: **Settings** (arriba) → **Pages** (menú izquierdo).
2. En **"Branch"**, elige **main** y carpeta **/ (root)** → **Save**.
3. Espera ~1 minuto. Aparecerá arriba tu enlace, algo como:
   **`https://TU-USUARIO.github.io/comunicador-rafael/`**

### 4. Abrir en el iPad
1. En el iPad, abre **Safari** y entra a ese enlace.
2. ¡Ya funciona completo! 🎉
3. Para dejarlo como app: **Compartir (cuadro con flecha ↑) → "Agregar a inicio"**.
4. Queda un ícono en la pantalla de inicio, a pantalla completa.

## 🔒 Privacidad
- El repositorio es **público**, pero solo contiene la app y los pictogramas
  (el nombre "Rafael" y el vocabulario). Cualquiera con el enlace exacto puede verla.
- **Las fotos que agregues** con el editor (mamá, papá, etc.) **NO se suben a GitHub**:
  quedan guardadas solo en el iPad.
- Si prefieres que sea privado del todo, se puede con GitHub Pro (de pago) o buscamos
  otra opción.

## ✏️ Usar y editar
- Toca **✏️** (arriba a la derecha) para entrar a modo edición.
- Agrega/cambia botones, pon **fotos reales** (📷) o pictogramas (🎨).
- Después de personalizar, usa **⬇️ Guardar copia** como respaldo.

## 🔄 Actualizar a una versión nueva
Si más adelante hay una versión nueva del archivo, subes el `index.html` nuevo a GitHub
(**Add file → Upload files**, reemplaza) y listo. Tus fotos y cambios en el iPad se conservan.
