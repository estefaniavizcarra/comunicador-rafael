# Comunicador de Rafael 🗣️

`index.html` es la app completa (funciona sola, sin internet una vez cargada).

## Para verlo en el iPad
1. Sube `index.html` a tu repositorio de GitHub (Add file → Upload files → Commit).
2. Abre el enlace de GitHub Pages en **Safari** del iPad.

## 🔊 Sobre el sonido (importante)
Quité el modo "pantalla completa" porque en el iPad **rompía la voz**.
Ahora, al abrirlo desde Safari (con la barra de dirección arriba), **el sonido funciona**.

Si aun así no suena, revisa en el iPad:
- Volumen arriba y **silencio apagado** (Centro de Control, ícono de campana 🔕).
- Que lo abras desde **Safari** (no desde un ícono de "pantalla completa").
- Toca un botón: el sonido solo sale al tocar (no automático).
